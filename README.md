MapleStory WZ Icon Viewer
=========================

A modern, dark-theme Tkinter application for browsing MapleStory WZ icons exported by the updated WZ Image Flattener.

This viewer supports the new nested JSON structure generated by the flattener and provides real-time PNG previews, type/category filtering, and a clean UI designed for MapleStory developers and private server creators.

------------------------------------------------------------
FEATURES
------------------------------------------------------------

• Supports new nested JSON format:

  Item/
    Cash/
    Consume/
    Etc/
    Pet/
    Unknown/

  Other types (Mob, Npc, Skill, etc.) remain flat.

• Dark theme matching the flattener:
  Background: #202124
  Text: #E8EAED
  Inputs and buttons: #303134, #3C4043

• PNG Preview:
  - Uses Tkinter PhotoImage (no Pillow required)
  - Auto-scale large PNGs
  - Graceful “File not found” handling

• Filtering & Navigation:
  - Type dropdown
  - Category dropdown (Items only)
  - ID substring search
  - Scrollable ID list
  - Metadata panel with type, category, ID, and full path

------------------------------------------------------------
REQUIRED FILES
------------------------------------------------------------

• icon_db.json  
  Generated by the new WZ Image Flattener.

• pngs_flat/  
  Root directory containing exported icons organized by type/category.

------------------------------------------------------------
USAGE
------------------------------------------------------------

1. Run wz_icon_viewer_gui.py
2. Click “Load icon_db.json”
3. Click “Select PNG root” and choose the folder containing PNGs
4. Select Type / Category / Filter
5. Click an ID to preview the PNG

------------------------------------------------------------
OPTIONAL: BUILD EXE
------------------------------------------------------------

The viewer is PyInstaller-friendly:

pyinstaller --noconsole --onefile wz_icon_viewer_gui.py

------------------------------------------------------------
REQUIREMENTS
------------------------------------------------------------

• Python 3.10+
• Tkinter (included with most Python installations)
• No external libraries required
