MapleStory WZ Image Tools — Icon Viewer
=======================================

A small utility for browsing MapleStory WZ icons that were exported and flattened by a separate tool.

This project is intended to be used together with:

- WZ Image Flattener (PNG + JSON generator)
  https://github.com/Souliolii/WZ-Image-Flattener
- WZ Icon Viewer (this repository)
  https://github.com/Souliolii/WZ-Image-Viewer

These tools are designed for MapleStory developers, private server maintainers, and data modders.

----------------------------------------
WZ Image Flattener (Required)
----------------------------------------
The **WZ Image Flattener** runs first in the pipeline.

Repository:
    https://github.com/Souliolii/WZ-Image-Flattener

It processes raw MapleStory WZ data and exports:

1) A fully flattened directory of PNG icons

   Example:
   pngs_flat/
       Item/Consume/2000000.png
       Item/Cash/05010000.png
       Mob/100100.png
       Npc/1012004.png

2) A structured JSON database (icon_db.json) describing all exported icons:

   {
     "Item": {
       "Consume": {
         "2000000": "Item/Consume/2000000.png"
       },
       "Cash": { ... },
       "Etc": { ... },
       "Pet": { ... },
       "Unknown": { ... }
     },
     "Mob": {
       "100100": "Mob/100100.png"
     },
     "Npc": {
       "1012004": "Npc/1012004.png"
     }
   }

The **icon_db.json** file created by the WZ Image Flattener is REQUIRED by the WZ Icon Viewer.
Without running the flattener first, the viewer will not have any data to display.

----------------------------------------
WZ Icon Viewer (This Project)
----------------------------------------
A dark-themed Tkinter application for browsing the PNGs and IDs generated by the WZ Image Flattener.

Repository:
    https://github.com/Souliolii/WZ-Image-Viewer

### ✔ No Python Required — EXE Available  
A **prebuilt executable (EXE)** is available in the GitHub Releases section.  
You do **NOT** need Python installed to run the viewer — simply download and launch the EXE.

Releases:
    https://github.com/Souliolii/WZ-Image-Viewer/releases

### Features
- Loads `icon_db.json` generated by the WZ Image Flattener  
- Uses the PNG root folder created by the flattener (e.g. `pngs_flat/`)  
- Browse icons by Type (Item / Mob / Npc / Skill / Etc.)  
- Browse by Category (Item only: Consume, Cash, Etc, Pet, Unknown)  
- Live Search — instant ID filtering as you type  
- PNG preview with auto-scaling  
- Handles missing PNGs gracefully  
- Dark UI matching the flattener  
- **Copy ID button** — copies the selected ID to clipboard  
  (useful for quest editors, scripts, rewards, documentation, etc.)

----------------------------------------
Required: Flattener Output Files
----------------------------------------
The WZ Icon Viewer depends on the output from the **WZ Image Flattener**.

You must have:

1) icon_db.json  
   - Generated by the flattener.  
   - Contains mappings of all exported items, mobs, NPCs, etc.

2) PNG root folder (e.g. `pngs_flat/`)  
   - Contains subfolders such as:
       pngs_flat/Item/...
       pngs_flat/Mob/...
       pngs_flat/Npc/...

The viewer uses icon_db.json + PNG paths to locate and display images.

----------------------------------------
Usage (WZ Icon Viewer)
----------------------------------------
### If using the EXE (recommended):
1. Download the EXE from the Releases tab  
2. Run it (no Python required)

### If running from source:
1. Install Python 3.10+
2. Run wz_icon_viewer_gui.py

### Inside the Viewer:
1. Click **"Load icon_db.json"**  
2. Click **"Select PNG root"**  
3. Choose a **Type**  
4. Choose a **Category** (Item only)  
5. Type into **Search ID** to filter results instantly  
6. Click an ID to preview  
7. Click **Copy ID** to copy the selected ID

----------------------------------------
Build EXE (Optional)
----------------------------------------
If you want to build your own EXE:

    pyinstaller --noconsole --onefile wz_icon_viewer_gui.py
